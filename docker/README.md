# Setup docker

## Install docker for your OS

Please refer to the official guides, for installing docker, to ensure the most up to date installation process.

- [Windows](https://docs.docker.com/desktop/setup/install/windows-install)
- [Linux](https://docs.docker.com/engine/install)
- [MacOS](https://docs.docker.com/desktop/setup/install/mac-install)

## Setting up docker-desktop

If docker-desktop was part of the installation process for your operating system, then open the program to start the docker engine.

## Run docker for this project

Open your terminal and navigate to to the [docker directory](https://github.com/Shazil0204/Smart-trafik-styring/tree/main/docker), inside this repository.

In the terminal, write "docker-compose up".

The docker containers will then start up.
